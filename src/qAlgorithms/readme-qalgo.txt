These files were added to implement a limited amount of functions derived from
the qAlgorithms project. Currently, this only includes the peak fitting function
(qpeaks_find() in the original project). To avoid conflicts and incompatibility,
a snapshot of the in-development version is supplied here. It is possible that
it still contains errors and should not be trusted blindly.

For updates on the source project, see github.com/odea-procject/qAlgorithms
